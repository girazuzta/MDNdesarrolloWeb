Las fuentes fueron descargadas en formato .ttf de font squirrel y luego se utilizó el web font generator
(https://www.fontsquirrel.com/tools/webfont-generator) de la misma página para crear el paquete con la familia de fuente
que está en esta carpeta. Por cada fuente hay dos tipos de archivos distintos debido a cuestiones de compatibilidad de navegadores.
